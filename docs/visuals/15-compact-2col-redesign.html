设计要点

  • 合并输入：Dash RangeSlider 自带 tooltip={"placement": "bottom", "always_visible": True}，直接在 bar 下方显示当前 Low/High 值。Low/High 文本输入框被彻底移除。
  • 双列布局：CSS Grid display: grid; grid-template-columns: 1fr 1fr;，两个指标 cell 共享一个外层 .range-row.double 边框。Similarity 单列场景使用 .range-row.single。
  • 状态视觉：.metric-cell.is-invalid / .is-warning 状态会传播到 cell 背景、handle 边框、track 渐变和 logic-compact 文本颜色。
  • 主题一致：全部使用项目 eda-theme.css 中的 CSS 变量（--accent-primary、--bg-secondary 等），支持深色 / 浅色主题切换。
  • 离线运行：仅依赖 dash 库，无网络资源、无 Bootstrap、无第三方字体。
