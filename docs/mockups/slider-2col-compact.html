交互逻辑
  1. 拖拽手柄 → 调节 low/high 边界 · 实时同步数学区间
  2. 悬停手柄 → tooltip 显示精确数值（Variant A 默认）
  3. 小数值标签（Variant B）始终显示当前值 · 随手柄移动
  4. 数值不可直接编辑 · 全部通过拖拽设置 · 减少视觉元素
  5. 约束违反：low > high → cell 背景泛红 + 滑轨渐变变红 + logic 行 ⚠
  6. 区间过窄（<5%）：cell 背景泛黄 + logic 行 ⚠ 提示
